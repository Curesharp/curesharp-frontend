<hr>

<h2>Passos para rodar a aplicação</h2>

- Baixar o projeto (.zip ou via git clone)
- Abrir o projeto no VsCode
- Instalar as dependencias do node-modules (npm install)
- Rodar o projeto (npm run dev)
